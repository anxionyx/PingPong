# PingPong
## A Ping Pong game made with rust to relax in terminal
> [!WARNING]
> Repo NOT Maintained,archieved in June 2026
### Installation
#### Method 1 : Manual Build
Run the following commands
```
git clone https://github.com/anxionyx/PingPong.git 
cd PingPong
cargo build --release
./target/release/PingPong
```
(Optional)
```
cp ./target/release/PingPong /usr/bin/pingpong
pingpong
```

Go to the release page and download the executable
